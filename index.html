Windows Instructions

  • Locate the downloaded Node.js installer (e.g., node-vXX.XX.X-x64.msi) on the computer and double-click it to start the installation process.
  • The setup wizard will guide through the installation. Click "Next" to continue.
  • Accept the license agreement and click "Next."
  • Choose the destination folder where Node.js will be installed or stick with the default location. Click "Next."
  • Select the components to install. It's recommended to keep the default options, including npm. Click "Next."
  • Choose whether to include the Node.js runtime in the system's PATH environment variable. Select "Automatically" for most cases. This will allow running Node.js from any command prompt. Click "Next."
  • Click "Install" to begin the installation process.
  • Once the installation is complete, click "Finish" to exit the setup wizard.

macOS Instructions

  • Locate the downloaded Node.js package (e.g., node-vXX.XX.X.pkg) on the computer and double-click it to start the installation.
  • The package installer will open. Click "Continue."
  • Read the license agreement, click "Continue," and then click "Agree" to accept it.
  • Choose the destination where Node.js will be installed. The default location should be appropriate for most users. Click "Continue."
  • Click "Install" to start the installation process. You may be prompted to enter the computer's password to proceed.
  • Once the installation is complete, click "Close" to exit the installer.

Linux Instructions

For Linux users, the installation process can vary depending on the distribution. However, most distributions provide Node.js in their package repositories.

  • Open a terminal on the Linux system.
  • Use the package manager specific to the distribution to install Node.js. For example:
    • On Ubuntu/Debian: sudo apt install nodejs
    • On Fedora: sudo dnf install nodejs
    • On CentOS/RHEL: sudo yum install nodejs
  • Verify the installation by checking the installed versions of Node.js and npm:
    • To check Node.js: node -v
    • To check npm: npm -v
